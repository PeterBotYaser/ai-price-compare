# AI Price Comparison Platform

## Vision
Die go-to Plattform für AI-Model-Preisvergleiche und günstige Zugriffswege.

## Problem
- User zahlen ~€100/Monat für AI-Tools
- Preise sind intransparent (verschiedene Routen = verschiedene Preise)
- Studentenrabatte werden oft übersehen
- Günstige Alternativen (wie Kimi über Synthetic) sind nicht bekannt

## Lösung
Eine Übersichts-Plattform, die zeigt:
1. **Preisvergleich** für alle großen Modelle (GPT, Claude, Kimi, etc.)
2. **Günstige Routen** (z.B. Kimi via Synthetic statt direkt)
3. **Deals & Rabatte** (Student, Startup, Bulk)
4. **Aktuelle Promotions**

## Monetarisierung
- Affiliate Links (OpenAI, Anthropic, Synthetic, etc.)
- Sponsored Listings
- Premium Features für Power-User

## Tech Stack (MVP)
- Static Site Generator (Next.js oder einfach HTML/CSS/JS)
- JSON-Datenbank für Preise
- Automatische Updates via API/Scraping
